# Shopora

An e-commerce store built with React, Vite, and Tailwind CSS.

## Description

Shopora is an online store where users can browse products, add items to their cart, and complete purchases.

## Built With

- React
- Vite
- Tailwind CSS
- React Router

## Getting Started

### Installation

```bash
git clone https://github.com/sofietoender/Shopora
cd shopora
npm install
```

### Running

```bash
npm run dev
```

## Features

- Product catalog with search functionality
- Detailed product page with reviews and discounts
- Shopping cart
- Checkout process
- Contact form with validation
